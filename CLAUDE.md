This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This repository contains French language learning notes and exercises. It is not a software project.

## Content Structure

`frenchlessons.txt` contains:
- French verb tenses (présent, passé composé, imparfait, futur, conditionnel, subjonctif, impératif)
- Conjugation tables for common verbs (être, avoir, aller)
- Vocabulary lists with English translations
- Lesson notes from classes (dated entries starting November 2025)
- Practice dialogues and exercises

## Working with this Repository

- When helping with French content, provide accurate grammar explanations and translations
- Lesson notes use a mix of French and English with emoji headers for organization
- New lesson entries should follow the existing date format: "Cours du [day] [date] [month] [year]"
